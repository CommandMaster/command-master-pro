# ❓ FAQ – Command Master Pro

#### 🔹 What is Command Master Pro?
Command Master Pro is a Blender addon that allows you to perform complex actions using simple text-based commands, saving time and speeding up your workflow.


#### 🔹 Is it free?
Yes ✅ Free for personal and commercial use.


#### 🔹 Does this addon work offline or online?
This addon works completely **offline** <br> 

- No internet connection is required  
- All commands run locally inside Blender  
- Fast and secure 


#### 🔹 Do I need coding knowledge?
No ❌
Just learn simple command patterns and you're good to go.


#### 🔹 Why should I use Command Master Pro?
Because it: <br>
- Saves time 
- Reduces repetitive steps 
- Makes Blender faster 
- Improves workflow efficiency 


#### 🔹 Is this addon beginner-friendly?
Yes! <br>
It is designed to simplify complex workflows into easy commands.

#### 🔹 Which Blender version is supported?
Blender 5.1.1+

#### 🔹 How do I install this addon?
Download the latest .zip file from Releases <br<
- Open Blender
- Go to Edit → Preferences → Add-ons
- Click Install and select the downloaded .zip file
- restart blender 
- Enable the addon by checking the checkbox ✅
Done!


#### 🔹 How do I open Command Master?
Press `Ctrl + Space` to open the command input panel.


#### 🔹 How do I run a command?
1. Open Command Master  
2. Type your command (e.g. `m.c 2 0 0 0`)  
3. Press `Enter`  



#### 🔹 Where can I learn more commands?
👉 Check the [documentation:](Commands/)


#### 🔹 Why is my command not working?
Check the following:
- restart blender 
- enable command master pro 
- Correct command syntax
- No extra characters

#### 🔹 Does it support Mac or Linux?
Currently, this addon is only supported on Windows. <br>
Mac and Linux support may be added in future updates.
