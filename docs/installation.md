# Installation Guide

Follow these steps to install **Command Master Pro** in Blender.

---

## 1. Download Addon

- Download the ZIP file of this repository
- Or clone using Git:

```bash
git clone https://github.com/CommandMaster/command-master-pro.git
```

## 2. Install in Blender

1. Open Blender
2. Go to Edit → Preferences → Add-ons
3. Click local Install  
4. Select the downloaded ZIP file
5. Click Install Add-on
6. restart blender

## 3. Activate Addon
Search: Command Master Pro <br>
Enable checkbox ✅


##  4. Start Using Commands

1. Open Blender  
2. Press **Ctrl + Spacebar**  
3. Open **Command Master Prompt**  

Now enter commands: <br>
👉 Quick cube add
```bash
m.c
```

👉 Quick wall create

```bash
m.c 2 0 0 0 0 0 0 4 0.1 2
```

## ❗ Troubleshooting
 Addon not showing? <br>
- Restart Blender <br>
- Check folder name (no spaces) <br>

 Command not working? <br>
- Make sure addon is enabled <br>
- Check syntax format 

## Update Addon
- Delete old version
- Install new ZIP again
