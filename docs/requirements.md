# 💻 Requirements

Make sure your system meets these requirements before installing.

## Blender Version
- **Minimum:** Blender 5.1.1
- **Recommended:** Latest stable release

## Operating System
- Windows 10 / 11

## Hardware
- No special requirements – runs on any system that runs Blender

## Python
- Python is built into Blender – nothing extra needed.

---

## ✅ Compatibility Check

To check your Blender version:
1. Open Blender
2. `Edit` → `Preferences` → `System`
3. Look for version number at the bottom

If you have 5.1.1 or higher, you're good to go!
