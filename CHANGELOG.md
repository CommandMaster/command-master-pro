# Changelog

All notable changes to this project will be documented in this file.

---

## [1.0.0] - Initial Release 🚀

### ✨ Added
- Cube creation command (`m.c`)
- Cylinder creation command (`m.cy`)  
- Sphere creation command (`m.s`)  
- Text creation command (`text`)  
- Create new collection command (`col.new`)  

### ⚙️ Core Features
- Command-based object generation system  
- Parameter-based syntax (size, location, rotation, scale)  
- Basic command parsing system  

### 📁 Structure
- Modular command folder system  
- Mesh-based organization (Commands → Mesh → Objects)  

### 📚 Documentation
- Initial GitHub docs setup  
- Command syntax documentation  
- Folder-based documentation structure

## 📌 Notes
- Project is under active development  
- Features and structure may evolve
